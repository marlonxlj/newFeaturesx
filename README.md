###XLJNewFeature新特性

####★★★"iOS App新特性"★★★

####支持pod导入

pod 'XLJNewFeature', '~> 1.0.2'

如果发现pod search XLJNewFeature 搜索出来的不是最新版本，需要在终端执行cd转换文件路径命令退回到desktop，然后执行pod setup命令更新本地spec缓存（可能需要几分钟），然后再搜索就可以了。
